Modulo de ayuda para el administrador del sistema


Para realizar el proceso de respaldar la base de datos, primero debes haber iniciado sesión como administrador para acceder a estas funcionalidades.

1. En el apartado de backup y Restore, en la barra de navegación tendrás acceso a estos procesos.
2. Para seleccionar respaldo, dirígete a respaldar base de datos.
3. Ingresa el nombre de la base de datos que se solicita en el formulario.
4. Dar clic en respaldo.
5. La aplicación descargara el archivo de respaldo con los datos que actualmente contiene tu base de datos. Este archivo debes guardarlo en caso de que lo necesites.
6. Este proceso se puede realizar cuantas veces el usuario estime conveniente.
